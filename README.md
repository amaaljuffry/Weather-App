## Setup

1. Clone & install dependencies:

```bash
git clone https://github.com/your-username/weather-app.git
cd weather-app
npm install
```

2. Add `.env` with your API keys:

```env
VITE_API_KEY=your_openweathermap_api_key
VITE_MAPTILER_KEY=your_maptiler_api_key
```

3. Run dev server:

```bash
npm run dev
```
